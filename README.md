# Tesla Social Media Analytics (Excel Project)

## 📊 Project Overview
This project analyzes Tesla's social media performance using Microsoft Excel, focusing on engagement trends, audience behavior, and content insights.

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Charts & Dashboard

## 📈 Key Insights
- Identified peak engagement periods
- Compared likes, comments, and shares
- Found high-performing content types

## 📂 Files
- Tesla_Social_Media_Analytics.xlsx

## 🚀 Conclusion
This project showcases Excel-based data analysis and visualization for business decision-making.
